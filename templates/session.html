Maan Documentation

Getting Started

Maan is a real-time collaborative code editor inspired by VS Code. Multiple users can work on the same project simultaneously, see each other's cursors, and communicate through chat.

Joining a Session

When you first access a session, you'll be prompted to enter your name. After entering:

  • Admins join immediately without approval
  • Anonymous users wait for admin approval before joining
  • Registered users join immediately if they created the session

File Explorer

The file explorer shows your project's file structure. Colored dots next to files/folders indicate active users.

Opening Files

  • Click any file to open it in the editor
  • Click folders with â–¶ to expand/collapse them
  • Multiple files can be open in tabs simultaneously

User Indicators ALL USERS

Colored dots show which files users are viewing:

  • Dot on file: User is viewing that specific file
  • Dot on folder: User is viewing a file inside that folder (when folder is collapsed)
  • Hover over dots to see username

File Management ADMIN ONLY

Right-click on files/folders or empty space to:

  • New File: Create a new file
  • New Folder: Create a new folder
  • Rename: Rename existing file or folder
  • Delete: Delete file or folder (cannot be undone)

Code Editor

The editor supports syntax highlighting for multiple languages and shows real-time collaboration features.

Keyboard Shortcuts

  • Ctrl/Cmd + S - Save current file
  • Ctrl/Cmd + F - Find in file
  • Ctrl/Cmd + H - Find and replace
  • Ctrl/Cmd + / - Toggle line comment

Real-time Collaboration

  • See other users' cursors as colored vertical lines with their names
  • Changes from other users appear instantly
  • Cursor position shown in status bar: Ln X, Col Y

Saving Files

  • Admins: Files save immediately
  • Users: Save requests require admin approval
  • Notifications appear in chat when files are saved

Active Users Panel

Click the icon in the activity bar to see all active users.

User Information

  • Each user has a unique color indicator
  • Small dot shows if user is viewing a file
  • Total user count shown in status bar

Jump to User Location ALL USERS

  • Click the button next to any user to jump to their current file
  • Button is disabled if user is not viewing any file
  • Automatically opens the file and switches to that tab

Kick Users ADMIN ONLY

  • Click to remove a user from the session
  • Kicked users must request to join again

Chat System

Real-time chat for team communication. Click in the activity bar to toggle chat panel.

Sending Messages

  • Type your message and press Enter to send
  • Sound notification plays when new messages arrive
  • System messages appear for user join/leave and file save events

Share Your Location ALL USERS

  • Type # alone and press Enter
  • Sends your current file location (e.g., user@path/file.js:L12C5)
  • Other users can click the link to jump to your exact position
  • Only works when you have a file open in the editor

Admin Features

Additional capabilities for session administrators.

Approval System

Admins receive notification cards in the top-right for:

  • Join Requests: When anonymous users want to join
  • Save Requests: When non-admin users want to save files
  • Sound notification plays for each request
  • Click "Approve" or "Reject" to respond

Session Management

  • Kick Users: Remove disruptive users from the session
  • File Operations: Full control over creating, renaming, and deleting files/folders
  • Direct Save: Files save immediately without approval

Project Dashboard

  • Access via "Back to Dashboard" from the navbar
  • View all your projects and their session URLs
  • Share session links with team members
  • Delete projects when no longer needed

Tips & Tricks

  • Session Persistence: Reload the page to reconnect to the same session
  • Color Coordination: Each user gets a unique color for easy identification
  • Context Menus: Right-click almost anywhere for quick actions
  • Tab Management: Click on tabs to close files you're done with
  • Status Bar: Shows session ID, user count, and cursor position
  • Collaborative Debugging: Use location sharing to guide teammates to specific code

Troubleshooting

Connection Issues

  • Refresh the page to reconnect
  • Check your internet connection
  • Ensure the session is still active

Can't See Other Users

  • Make sure you've been approved by the admin
  • Check the Users panel to see who's online
  • Users must be viewing the same file to see their cursors

File Changes Not Appearing

  • Verify your save request was approved (if not admin)
  • Check if the file was saved by another user
  • Reload the file by closing and reopening it
